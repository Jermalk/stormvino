**Q: How to add the SearxNG JSON format param permanently to the docker run command?**

A: Copy the already-edited settings file out of the container first, then mount it on next run:

```
docker cp searxng:/etc/searxng/settings.yml ~/searxng-settings.yml
```

Recreate with bind mount:
```
docker run -d --name searxng -p 8080:8080 -v ~/searxng-settings.yml:/etc/searxng/settings.yml searxng/searxng
```

docker-compose equivalent:
```yaml
services:
  searxng:
    image: searxng/searxng
    ports:
      - "8080:8080"
    volumes:
      - ~/searxng-settings.yml:/etc/searxng/settings.yml
```

**Q: I need to remove existing container first, how to?**

A:
```
docker rm -f searxng
```
`-f` forces removal even if running — no need to stop first.
